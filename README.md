# 🛒 LumiCart Product Analytics

An end-to-end Product Analytics portfolio project for a fictional e-commerce platform using SQL, Python, Power BI, statistics, and machine learning.

## Project Status

🚧 This project is currently under development.

## Business Goal

The goal of this project is to understand customer behavior, identify conversion and retention problems, and provide data-driven recommendations for an e-commerce business.

## Main Analyses

* E-commerce funnel analysis
* DAU, WAU, and MAU analysis
* Cohort and retention analysis
* A/B testing for a new Buy Now button
* Customer segmentation using RFM
* Churn prediction using Logistic Regression
* Product KPI dashboard in Power BI

## Technology Stack

* SQL
* Python
* Pandas
* Jupyter Notebook / Google Colab
* Power BI
* Scikit-learn
* Git and GitHub
